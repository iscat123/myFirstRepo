linux目录结构：	/sbin	s表示Super User,存放的是系统管理员使用的系统管理程序 \
				/bin	是Binary的缩写，这个目录存放最常用的命令(/usr/bin、/usr/local/bin) \
				/home	存放普通用户的主目录 \
				/root	该目录为系统管理员，超级权限的用户主目录 \
				/lib	动态链接共享库，几乎所有的应用程序都需要的共享库 \
				/lost+found	这个目录一般是空的，系统非法关机后，这里就存放一些文件 \
				/etc	所有的系统管理所需的配置文件和子目录，比如安装mysql数据库 my.conf \
				/usr	用户的应用程序和文件放这个目录，类似与Windows下的program files目录 \
				/boot	存放linux启动所需的核心文件，包括一些链接文件和镜像文件 \
				/proc	【不能动】虚拟目录，系统内存映射，访问这个目录获取系统信息 \
				/srv	【不能动】service缩写，存放服务启动之后需要的数据 \
				/sys	【不能动】该目录安装了2.6内核中新出现的一个文件系统sysfs \
				/tmp	存放一些临时文件 \
				/dev	类似于windows的设备管理器，把所有的硬件用文件的形式储存 \
				/media	linux系统会自动识别一些设备，例如U盘、光驱，这些设备会被挂载到这个目录下 \
				/mnt	系统提供该目录让用户临时挂载别的文件系统，我们可以将外部的存储挂载在/mnt/上，然后进入该目录查看内容 \
				/opt	给主机额外安装然间所摆放的目录。如安装mysql数据库可以放到这个目录下 \
				/usr/local	这个另一个给主机额外安装软件所安装的目录。一般是通过编译源码方式安装的程序 \
				/var	这个目录存放不断扩充的东西，习惯将经常被修改的目录放在这个目录下。包括一些日志文件 \
				/selinux	SELinux是一种安全子系统，它控制程序只能访问特定文件。
				
vim/vi：	正常、插入和命令行三种模式 \
			复制 [yy] 复制5行 [5yy] \
			粘贴 [p] \
			删除 [dd] 删除5行 [5dd] \
			查找关键字 [/关键字] ,回车查找，[n]就是查找下一个 \
			设置行号 [:set nu] \
			取消行号 [:set nonu] \
			回到首行 [gg] \
			回到末行 [G] \
			撤销 [u] \
			光标移动到20行 [20 shift+g]

内存数据同步到磁盘：sync	一般关机重启先执行这个（虽然现在关机和重启会自动同步）
关机和重启命令：shutdown -h now	立刻关机 \
				shutdown -h 1	一分钟后关机 \
				shutdown -r now	现在重启 \
				halt			关机 \
				reboot			重启 \

用户登入和注销命令：登入或切换用户：	su [用户名]	 \
					注销：	logout/exit	

用户管理命令：添加用户：	1.useradd [用户名] \
							2.useradd -d [目录] [用户名] 添加用户并指定目录 \
							3.useradd -g [用户组] [用户名]添加用户并指定用户组 \
			  指定或修改密码：	passwd [用户名] \
			  删除用户：	1.userdel [用户名]	删除用户但保留家目录 \
							2.userdel -r [用户名]	删除用户和家目录 \
			  修改用户的组：usermod -g [用户组] [用户名] \
			  查询用户信息：id [用户名] \
			  查看当前用户：whoami \
			  
			  添加用户组：groupadd [组名] \
			  删除用户组：groupdel [组名] \
			  /etc/paddwd文件：用户的配置文件，记录用户的各种信息 \
			  /etc/shadow文件：口令配置文件 \
			  /etc/group文件：组的配置文件
			  
帮助指令：man [命令] \
		  help [命令]

文件目录指令：	创建多级目录：mkdir -p [多级目录] \
				删除目录：1.rmdir [目录名] 注：只会删除空目录\
						  2.rm -rf [目录名] \
				创建空文件：touch [文件名称] \
				复制：cp -rf [source] [dest] 强制覆盖且递归 \
				查看文件：1.cat -n [文件] 显示行号 \
						  2.more [文件] 分屏显示 \
						  3.less [文件] 分屏显示且边读边加载 \
						  4.head -n [文件] 不指定n时，默认显示文件前10行(例：head -n 12 a.txt)\
						  5.tail -n [文件] 不指定n时，默认显示末尾10行(tail -f a.txt实时监控) \
				重定向：echo [信息] > [文件] \
				追加  : echo [信息] >> [文件] \
				软连接：ln -sf [原文件] [软链接名] (删除用rm命令) \
				查看历史执行的指令：history 后面加数字就是近几个指令 \
				执行历史编号为5的指令：!5 \

时间日期指令：显示当前时间 date \
			  设置时间 date -s [2024-08-27 22:25:00] \
			  查看日历 cal [年月] 不加参数默认当前年月 \
			  
搜索查找指令：查找：find [目录] [-name/-size/-user] [文件名/+200M/root] \
			  搜索文件位置：updatedb + locate [文件] \
			  查看指令位置：which [指令] \
			  过滤查找：grep [-n/-i/-r] [查找内容] [文件] n:行号，i:忽略大小写,r:搜索文件 \
						cat a.txt |grep -ni "hello" \

压缩解压指令：gzip hello.txt  得到hello.txt.gzip 只能压缩文件\
			  gunzip hello.txt.gz \
			  zip [-r] [xxx.zip][文件或目录] -r:递归目录 \
			  unzip [-d] [指定目录] [xxx.zip] \
			  tar -zcvf xx.tar.gz [文件] \
			  tar -zxvf xx.tar.gz -C [目录]
